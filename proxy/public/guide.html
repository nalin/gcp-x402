BigQuery · no Google Cloud account

Query public data from your agent.

Get answers from Google BigQuery's ~200 public datasets — census, weather, crypto, Hacker News, and more — paying a fraction of a cent per query in stablecoin. Your agent handles everything.

Install — just tell your agent
install the skill at https://gcp-x402-975410367881.us-central1.run.app/skill
That's the whole install. Your agent fetches the skill and learns how to use the tool — no setup, no keys.
  1. Install the skill

    Tell your agent: install the skill at https://gcp-x402-975410367881.us-central1.run.app/skill

    Prefer to do it by hand?

    Run this in the project you want to enable (installs it just there):

    mkdir -p .claude/skills/bigquery-public-data && \
curl -fsSL https://gcp-x402-975410367881.us-central1.run.app/skill \
  -o .claude/skills/bigquery-public-data/SKILL.md
  2. Fund the wallet (one time)

    The tool makes a small crypto wallet for the project and needs a little USDC on Base to pay with. Ask your agent:

    what's my gcp-x402 wallet address?

    Send that address some USDC:

    • Testing (free): faucet.circle.com → choose Base Sepolia → paste the address. A dollar or two lasts a long time.
    • Real use: send USDC on Base from any wallet or exchange.

    No ETH/gas needed — just USDC. Each project keeps its own wallet.

  3. Ask away

    Ask normal questions — the agent writes the SQL, shows the price, pays, and answers:

    What were the 10 most popular baby names in California, all-time? Use BigQuery. From the public Hacker News data, top 5 stories by score in 2015? Estimate what that query would cost before running it.

What it costs

Price scales with how much data a query scans (≈ $6.25 / terabyte), with a tiny floor.

QueryTypical cost
A focused query on a normal table$0.002 (the minimum)
Scanning ~1 GB~$0.012

The tool won't auto-pay more than $1.00 on a single query unless you raise the limit, and you're only charged for queries that actually succeed.

Troubleshooting

“host_not_allowed” / blocked when installing or querying

Your agent runs in a locked-down environment that only allows certain outbound hosts. Ask whoever manages it to allow these:

HostNeeded for
gcp-x402-975410367881.us-central1.run.appinstalling the skill + running queries
github.com, codeload.github.com, registry.npmjs.orginstalling the tool itself
sepolia.base.org (or mainnet.base.org)checking your wallet balance

“Insufficient funds” or a payment error

Your wallet is low. Ask “what's my gcp-x402 wallet address?”, send it some USDC, and retry.

“Only public data is queryable”

Correct — only Google's bigquery-public-data datasets, read-only (SELECT). That's still the full public catalog: census, weather, crypto, GitHub, Hacker News, Stack Overflow, and ~200 more.

Good to know
You never need a Google Cloud account, an API key, or a credit card. The only thing you fund is the small USDC wallet — and only for the queries you actually run.

Power users: MCP instead of a skill

If your client supports MCP, register the tool directly for structured tools (bigquery_query, bigquery_estimate, wallet_info, list_public_datasets):

claude mcp add gcp-x402 \
  --env PROXY_URL=https://gcp-x402-975410367881.us-central1.run.app \
  -- npx -y github:nalin/gcp-x402

Same wallet, same per-query pricing.
